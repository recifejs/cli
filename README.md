## Quick Overview

**Create new project**

```
npx recife-cli project my-project-name
```

**Start the project**

```
cd my-project-name
npm start
```

## How to create a recifejs project?

There are a few ways to create a Recife project.

### Yarn

```
yarn create recife-cli my-project-name
```

### NPX

```
npx recife-cli project my-project-name
```

### NPM

```
npm init recife-cli project my-project-name
```
